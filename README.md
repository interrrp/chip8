# 🕹️ CHIP-8 Emulator

A tiny [CHIP-8](https://en.wikipedia.org/wiki/CHIP-8) emulator.

## 🧑‍💻 Usage

### From source

#### Prerequisites

- [Rust Nightly](https://rust-lang.org)

#### Steps

1. Clone the repository

   ```sh
   git clone https://github.com/interrrp/chip-8
   cd chip-8
   ```

2. Run

   ```sh
   cargo run --release -- <args>
   ```
